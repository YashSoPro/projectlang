About ProjectLang

ProjectLang is a unique programming language designed to simplify coding through natural language processing. With ProjectLang, you can translate your commands into executable code effortlessly.

How It Works:

  1. Input your commands in natural language.
  2. Use the editor to create files with a .dumb extension.
  3. Compile and run your code seamlessly.
